# Retail Sales Analysis Dashboard

## Project Overview

This project analyzes retail sales data containing 9,994 transactions, 5,009 orders, and 793 customers.

The objective was to identify sales trends, profitability drivers, customer behavior, and regional performance using Python, PostgreSQL, and Power BI.

---

## Tools Used

* Python (Pandas)
* Google Colab
* PostgreSQL
* Power BI
* Excel

---

## Project Workflow

Dataset → Data Cleaning (Python) → SQL Analysis (PostgreSQL) → Dashboard Development (Power BI)

---

## Key Business Insights

* Technology generated the highest overall profit.
* West region contributed the highest sales revenue.
* Consumer segment accounted for more than 50% of total sales.
* Furniture showed comparatively lower profitability.
* Higher discount levels were associated with lower profits.
* Several products consistently generated negative profit.
